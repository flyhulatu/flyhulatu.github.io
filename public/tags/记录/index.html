健身的一些优先级，新年第一天

健身的一些优先级 今天练背了，这个过程中产生了一些思考。 如果你想接触力量训练，那么以下这些东西，是我认为你必须要「坚守的基础原则」，有点像数学公理——任何科学必须要遵守。 01 动作标准大于一切 想要健身，无疑是想给某些部位的肌肉群施加刺激，而后让受刺激的部位通过营养补充和睡眠后，得到修复，从而变得更加强壮。让我们身体的某些部位更立体、更凸显，穿衣和日常姿态更美观。 但如果你过分的追求「刺激」，用了不合适的重量、用着变形的动作、刺激着其他的肌肉群，那么很有可能，你会有代偿，甚至受伤！ 很多人会狡辩说：那这样的话，我也连到了代偿的肌肉群呀，也是一种刺激和进步。Nonono！这样只会让你的目标肌群和代偿肌群都练不好。 因为你首要目的项链目标肌群，说明动作模式是针对该肌群的，但由于重量不合适或者动作理解不到位，你调动了一些代偿肌肉，过分的借力和改变动作，其实代偿肌肉也练不好，因为你原本的动作模式就不是专门练代偿肌肉群的动作，我讲明白了吧！？ 就好像一个人他去相亲时没有匹配到对象，他反倒对身边人说他想过单人生活、单人生活更适合自己。那他单人的生活就一定过的多姿多彩、自由洒脱吗？大概率并不是的吧！要不然他要不要看看自己在做什么。 02 找准发力感是加重量的基础 动作准了就一定有好的发力感嘛？并不是的，有些人的发力感需要「自己去自适应」。 为什么？因为有的人的身体结构比较独特。 比如连杠铃划船，有的人手臂超级长，那么他可能杠铃的位置就需要摆到膝盖以下，划船的时候背部才会有更好的发力感；比如哑铃上斜卧推，有的人手臂很短，那么他很容易的就做到全程小臂垂直地面，因为他们的阻力臂比正常人要小不少，就比较节约能量。 我们只有在健身的时候，学会标准动作模式，调整、找到适合自己的「发力感」，如此，我们才能在这个感觉下，维持动作的稳定、加大发力的感觉和肌肉的刺激，这才能慢慢的上强度，也是正确的上强度。 03 为什么有些肌肉群难以找到发力感 有些动作比较好学习，肌肉的发力感也很容易找到，比如二头弯举、比如箭步蹲。而有些动作，肌肉总是很难调动、很难有良好的发力感，比如哑铃划船。 为什么不同的动作和肌肉群的调动难度不一样？ 我在凯胜王的视频里听到了凯哥说的一句话：肌肉调动的难易程度，与我们日常生活的动作频率，成反比。啥意思？就是说，我们有些动作日常生活中经常做，自然而然的那些肌肉群就比较活、比较好调动；而有些动作很少做，压根就不做。所以那一部分的肌肉很弱、也就难调动，难有感觉。 这个概念，非常的受用。很明显，蹲起的动作，我们很常见，所以深蹲注意一些细节，我们很容易找到下肢的发力感。但划船的动作，除了专业的划龙舟选手，我想大部分人是用不到「划」这个动作的，所以练习背部的动作比较难掌握，背部的肌肉比较难有发力感和刺激感，很正常，因为它们用的就很少。 如果你感觉学习一个动作，找不到发力感，或者有很严重的代偿、目标肌群外的部位先累了。听我的，果断降重量，把动作模式做熟悉、发力感找准，再去加重量，这是个必然的过程。 综上，是我今晚在练背部，做杠铃划船时的一些感受。因为我觉得我已经学习了很多次正确的划船应该怎么做，但视频拍出来感觉总是哪里不对、背部也很难夹紧。重量一降，效果立马显著提升。 希望对大家有帮助。 以下是我的一些今日片段，和大家分享： 2026的第一天，大家都是怎么过得？ 我陪小张吃了好吃的午饭，逛了商场，买了新鞋，看到了cosplay，做了力量训练，丰富充实又满足的一天。

2025年度总结

2025年终小结 去年的年终小结在这里： 我的 2024 年度小结 一眨眼，今年，我研二了。 讲心里话，我是个很爱给自己做总结的人，但回看过去的一年，又觉得它难以书写——我何德何难，仅靠着一些书面文字就可以总结完自己丰富多彩又难以被定义的一年。 我很复杂，难以简介！ 01 概括 终归还是要给自己的 2025 画上一个句号，用一个字来概括我的2025，我想非「淡」字莫属。 我是个很「淡」的人，可能很多人没有概念。但事实就是如此，我是个非常淡的人，倒不是我刻意让自己表现的不焦虑、没有未来担忧，而是我从心底里「感受不到很多基于眼下现状而延伸的未来焦虑」，我是真的没有这种感觉。甚至于这个过程中都不存在我告诫自己担忧没用、改变不了什么这样的自我训诫和心理暗示，是压根我就没有那种「过分焦虑未来」的感觉和意思，我……我，我气愤呀，哈哈哈！ 貌似我这样的人在很多人眼中是不求上进、不懂事的一类人，你可以说我是「混日子」的人！Okay！没关系，我接受的。我也想让自己有未来意识，但抱歉，真的在我脑子里没有这个概念的空间和余地，我就是一个很淡很淡的人。淡到没有过多的未来想法，淡到很多「有竞争性的评奖和活动」我都没有丝毫兴趣，淡到可以很长一段时间我都把精力放在自己想做的事情上、而不是重要的事情上……。 2025年，是图图度过的淡淡的一年！ 02 收获 001 科研 作为一个工科研究生，完成课题、达到毕业条件永远是我的本职工作。 几经周折后，我的课题最终在今年7月份定下来了，通过这一段时间的文献阅读以及近期学习综述写作，我对「沙棘and沙棘果渣」这一领域的了解，愈发深入。 这个学期我给自己定下的科研任务：在寒假前能够找到我理想中的「预处理」方法，这是贯穿我毕业课题的东西；以及综述文章投稿。我有信心可以完成。完成后，我会把经验分享给大家，希望给你们带去一些帮助。 002 自媒体 这一年写出了不少热门的文章，使得我获得了前所未有的关注量和流量推荐。当然，伴随着流量一同出现的，往往是还不错的财务支持，很开心的一年，谢谢各位衣食父母，谢谢你们！ 公众号里，个人比较满意以下这些文章。 除公众号外，我在少数派这个平台，也有一些文章被推上首页精选社区： ![1]( https://cdn.jsdelivr.net/gh/flyhulatu/img@main/uPic/CleanShot 2025-12-29 at 23.36.38@2x.png ) ![2]( https://cdn.jsdelivr.net/gh/flyhulatu/img@main/uPic/CleanShot 2025-12-29 at 23.36.09@2x.png ) ![3]( https://cdn.jsdelivr.net/gh/flyhulatu/img@main/uPic/CleanShot 2025-12-29 at 23.35.39@2x.png ) ![4]( https://cdn.jsdelivr.net/gh/flyhulatu/img@main/uPic/CleanShot 2025-12-29 at 23.35.14@2x.png ) 小红书，我也做了一点东西，试探一下这个平台的推流机制，为日后做铺垫： ![xhs]( https://cdn.jsdelivr.net/gh/flyhulatu/img@main/uPic/CleanShot 2025-12-29 at 23.42.20@2x.png ) 希望可以明年达到500粉，毕竟公域流量的平台，只要你有一个好的内容被流量砸中，就会有庞大的「滚雪球效应」。 此外，我的个人博客也被BlogFinder收录，以后我在我的个人博客更新的每一篇文章，都能被BlogFinder主页展示，欢迎大家浏览。 ![blog]( https://cdn.jsdelivr.net/gh/flyhulatu/img@main/uPic/CleanShot 2025-12-29 at 23.34.46@2x.png ) ...

我是个爱折腾的人

我是个爱折腾的人 不喜欢现成的东西 前段时间，我在调试更新一台旧 Windows 电脑，给它加了内存条、固态硬盘、重装了系统，最后打开 Win10 操作系统界面的时候，那种成就感和满足感是没有这种动手能力的人难以体会的。 后面我发现这个电脑的显示屏幕效果超级差，不单单是分辨率和帧率的问题，就是在屏幕上蒙上了一层纱布、有一种「云雾笼罩」的 feel。为了让它更好的发光发热，我反手就购入了一块 IPS 1080P 的高清屏，到手后立即就给它安上了，效果我还是很满意的。虽然最终的显示效果还远达不到我 MacBookPro 的细腻通透，但总算是能看了，而且也不累眼睛，就足够了。 这台电脑的系统我也是经常捣鼓，我先表明态度：对于电子产品，尤其是每天要长时间接触的电子产品，我是有「审美要求」的——这个产品，要让我有想要打开它的冲动。无论是硬件还是软件，都是如此。就拿 Windows 的操作系统来说，我个人是 Win7 的死忠粉——拟物化立体图标、磨砂透明效果以及高效直观的操作逻辑……，总之，这是个让我有打开冲动的系统，甚至没事时我都愿意点开各种软件窗口去欣赏一下它的动效、磨砂窗口。并且由于系统并不冗余臃肿，反应速度也飞快，那真是一种丝滑的体验。 但最后我还是没有装载 Win7 系统，因为微软公司已经在 19 年停止对它的支持了。换句话说：这个系统未来使用会出现各种问题，驱动上、补丁上、软件支持上的种种问题……，所以我换上了同样前不久宣布停止支持的 Win10 系统。 Win10 系统有好多个版本，从 15-25 年，10 年间 Win10 有无数个版本诞生、更替。因为我这台 Windows 电脑 Lenovo G40-80 是 2015 年的电脑，最新的系统我肯定是不考虑，而且 Bug 很多，虽说换上了 16G 双通道的内存，还有 2 块 SSD，但鉴于 i5-5200u 孱弱的CPU 性能，我还是希望它能够流畅运行系统和一些工科科研软件，因为我的 Mac 有挺多软件压根就不支持。 我尝试过 Win10 2015 版，也尝试过 Win10 LTSC 2019 版（被网友戏称为“老坛酸菜”养老版本）、还尝试过 Win10 LTSC 2021 版，但实际体验给我的反馈是，都不如 Win10 22H2 这个大版本号稳定流畅。虽说这个系统的安装包和系统文件比其他几个版本大，但胜在完备，实际使用的流畅程度和稳定程度我认为都高于其它的版本，所以装好这个系统版本后，我紧接着给它检查更新了一番，安装适配这个系统的补丁和驱动，最后下载了一些常用的办公软件（Office 和 WeChat等）和科研软件（SPSS 和 Origin），最后，我的这台电脑，本就打算停在这个版本养老了。 ...
