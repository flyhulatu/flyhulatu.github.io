我是个爱折腾的人#

不喜欢现成的东西#

前段时间，我在调试更新一台旧 Windows 电脑，给它加了内存条、固态硬盘、重装了系统，最后打开 Win10 操作系统界面的时候，那种成就感和满足感是没有这种动手能力的人难以体会的。

后面我发现这个电脑的显示屏幕效果超级差，不单单是分辨率和帧率的问题，就是在屏幕上蒙上了一层纱布、有一种「云雾笼罩」的 feel。为了让它更好的发光发热，我反手就购入了一块 IPS 1080P 的高清屏，到手后立即就给它安上了，效果我还是很满意的。虽然最终的显示效果还远达不到我 MacBookPro 的细腻通透，但总算是能看了，而且也不累眼睛，就足够了。

这台电脑的系统我也是经常捣鼓，我先表明态度：对于电子产品，尤其是每天要长时间接触的电子产品，我是有「审美要求」的——这个产品，要让我有想要打开它的冲动。无论是硬件还是软件，都是如此。就拿 Windows 的操作系统来说，我个人是 Win7 的死忠粉——拟物化立体图标、磨砂透明效果以及高效直观的操作逻辑……，总之，这是个让我有打开冲动的系统，甚至没事时我都愿意点开各种软件窗口去欣赏一下它的动效、磨砂窗口。并且由于系统并不冗余臃肿，反应速度也飞快，那真是一种丝滑的体验。

但最后我还是没有装载 Win7 系统，因为微软公司已经在 19 年停止对它的支持了。换句话说：这个系统未来使用会出现各种问题，驱动上、补丁上、软件支持上的种种问题……，所以我换上了同样前不久宣布停止支持的 Win10 系统。

Win10 系统有好多个版本，从 15-25 年，10 年间 Win10 有无数个版本诞生、更替。因为我这台 Windows 电脑 Lenovo G40-80 是 2015 年的电脑，最新的系统我肯定是不考虑，而且 Bug 很多，虽说换上了 16G 双通道的内存，还有 2 块 SSD，但鉴于 i5-5200u 孱弱的CPU 性能，我还是希望它能够流畅运行系统和一些工科科研软件，因为我的 Mac 有挺多软件压根就不支持。

我尝试过 Win10 2015 版，也尝试过 Win10 LTSC 2019 版（被网友戏称为“老坛酸菜”养老版本）、还尝试过 Win10 LTSC 2021 版，但实际体验给我的反馈是，都不如 Win10 22H2 这个大版本号稳定流畅。虽说这个系统的安装包和系统文件比其他几个版本大，但胜在完备，实际使用的流畅程度和稳定程度我认为都高于其它的版本，所以装好这个系统版本后，我紧接着给它检查更新了一番，安装适配这个系统的补丁和驱动，最后下载了一些常用的办公软件（Office 和 WeChat等）和科研软件（SPSS 和 Origin），最后，我的这台电脑，本就打算停在这个版本养老了。

以为这样就结束了吗？

并不是的。前面不是说了我对 Win7 系统有执念嘛，那是最符合我个人审美的一代经典系统，我初中和高中接触的电脑就是安装的这个系统，我还是很舍不得它。

于是乎，我又开启了新的折腾之旅：我去 Google 上搜索各种各样的 「Win10 美化成 Win7」 的教程和帖子，看了挺多 Bilibili 和 Youtube 上的教程，还下载了不少魔改的工具、软件、图标包和 Win7 主题等等。在最后安装 Win7 7tsp 图标的时候，我以为就要大功告成了，结果，它由于 WIFI 连接不稳定，黑屏了。

所以，我捣鼓美化的心，彻底的沉寂了，哇凉哇凉的呀。于是乎我又重装了一次系统，把设置以及常用的软件都安装好，最后也就不再折腾了。

我的 MacBookPro 也被我折腾过不少：首先就是系统版本的选择问题。

我先给使用 Mac 电脑的用户一个结论，如果你的芯片是 Inter 的老款电脑（2020 年以前），最适合的系统是 Macos 12.5，再早的系统就没有必要安装了，因为软件支持极其不友好。换句话说，如果你的电脑运行 Macos 12.5 都感觉吃力、不流畅，那你大概率是需要换电脑了。M 系列芯片的电脑（2020 年及以后），无论你是 M1 还是 M5，我都推荐安装 Macos 15.7.2（Macos 15 大版本的最新版），这个系统对于 M 系列芯片的优化很棒，且对于软件有着最好的支持，不用去更新 Macos 26，因为这个系统还在优化阶段，不稳定、有 Bug，等到版本稳定下来再尝鲜，也不迟。

我之所以敢和大家分享这个结论，是因为我自己从 Macos 11 BigSur 系统到 Macos 15 系统，都有使用过。而且也有测试过一些常用软件，但最后我还是回到了 Macos 15.7.2 的版本，因为我是 HomeBrew 的重度使用患者，并且喜欢 Macos 最新的动态壁纸，再加上动效和稳定性都还不错，而且我的很多终端工具都配置好了，真的不想继续折腾了。

就在这儿待着吧！

给师姐捣鼓电脑#

师姐是个电脑小白，符合大部分男生对女生的刻板印象的那种小白，甚至还要小、还要白（师姐，别笑）。

师姐原来的电脑比较老旧、卡顿，嗯……，肉眼可见的应该是比我那台旧 Windows 还要旧，有点影响使用体验了。她的新电脑是一个联想拯救者 32G + 2T 的配置，原装 Win11 系统，其实我从心里更想给师姐安装 Win10 的最终版系统 22H2。首先我觉得符合师姐的使用习惯，她用这个系统很多年了，习惯了 Win10 的操作逻辑；而且，Win10 已经终止支持了，说明这个系统不再有额外的更新服务，会少很多因为更新、重启发生的问题，毕竟师姐在电脑使用上，那可是纯洁的像白纸一样，不希望她在使用电脑时出现问题；最后，Win11 系统我没用过，嘿嘿，师姐的电脑如果出现了问题，我也很难主动解决，会显得我很呆（开玩笑）。

由于我没有带我的系统安装 U 盘，所以我就在师姐电脑自带的 Win11 系统上给师姐的固态硬盘分区了，分好区后用「一键部署工具」给师姐装了一个 Office 专业增强版 2024，然后给她激活，可以正常的使用。

其实没花多少时间，所以也不需要什么感谢，嘿嘿。要说请我喝 Coffee☕️ 嘛！嘿嘿，不需要，真的不需要。

其它的折腾#

我爱折腾这件事情，并不只体现在折腾电脑上。

我还很喜欢试探手机的系统版本号，我现在的手机 OriginOs 系统，我就一直觉得 OriginOs 4 比 OriginOs 5 稳定且省电，要不是为了谷歌框架，我是真的不会为了一些用不上的小功能而去更新一个大的系统版本，甚至，我的很多软件都是「老年版」，别笑，我没有开玩笑。

小时候，我用积木累的玩具拼接出了一辆属于自己的半挂卡车模型，但它没有现实生活中，欧卡的炫酷灯效，于是乎我把家里完好的「万能充」（就是那个可以夹着手机电池充电的玩意儿）上的七彩灯泡拆下来了，然后没有合适的电源，又把很多遥控上的电池挖空，最后我在那辆卡车模型上配置了一个灯光系统，真的很炫酷。收获成品的那一刹那，我是洋洋得意且非常满足的，但迎接我的，无疑是一顿教训，毕竟，把那么多没有坏的东西，提前宣布报废返厂。

折腾的乐趣#

我还爱折腾，说明我还有一点空闲。这里指的「空闲」，不是真的说我没有其他重要的事情催促我，相反，成年人，你很难说自己绝对自由空闲，因为某种意义上来讲，你可以做一切可以让你更加正向、更有竞争力的事情。

但我不想那样，我甚至会在压力很大的时候，去折腾一些我喜欢做的事情，如此，我才能感知到：原来我还有热爱！

未来的日子，我可能还是会常常被人认为是一个「不务正业」的男生，但没办法，你认为我「无所事事」的状态，恰好是我真正逍遥自在的片刻。
