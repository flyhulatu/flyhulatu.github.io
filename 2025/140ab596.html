写作那些事儿

摘要：写东西这事儿，最好明白自己为什么而写、怎么去写、用什么工具写、从哪儿找灵感。我在这篇文章里会分享我自己的经历和心得，希望能给其他写作者一些启发。

01 自媒体的内核

自己刚开始写东西的时候（公众号），其实经过了很多的心理斗争——我写的东西有没有价值？我会不会哪天就没东西可以写了？会不会有人嫌弃我的逻辑表述？我的排版会不会很差劲？熟人看到了会不会在心里笑话我……

但事实告诉我：很多东西我们之所以开始去做，要么是有一些难以承担的后果，要么是有太多想要开始的热情和冲动，超越了计较得失利害的理性！

显然，我的选择属于后者——我不去写东西的话，没有什么需要承担的后果，反倒每天、每周、每月会多出一些时间来做所谓的“别的有意思的事情”！

所以我写东西的开始，只是因为真的想写——想去观察、想去记录、想去表达、想去锻炼！！

这个选择的过程中我也明白了：理性不是一味的去否认我们的感受，而应该帮助我们去理解感受。

一直到后来，我的心态和对写东西的理解都有了改变——以前写东西总想教会别人一些什么，现在写东西只是想分享一些内容——自媒体的核心就是「分享」！

视频博主是用自己的镜头去记录、分享自己的生活！

文字博主是用自己的观察和思考去记录、表达自己的生活！

播客博主是用对话的多元视角去融合自己的生活！

……

所以，如果你也想做点什么、分享记录点什么，别去想变现、主题、教育和优越感……这些字眼。问自己一句——我真的真的有哪些东西很想很想和别人分享——一句话、一本书、一个餐厅、一件好物、一部电影、一个碎碎念……，如果你有这个念头，那就开始吧！

做自媒体很简单——分享就是了！

如果你没有太好的内容和深耕的领域想尝试，我希望你可以从尝试写日记开始。

02 如果想写日记

写东西，很多人都是从写日记开始的，这个过程有很多好处：

  • 让自己写出一些东西，没有太多的限制和要求；

  • 让自己保持写东西的习惯，不用担心灵感枯竭等等困境；

  • 写日记囊括了很多好习惯——短暂思考、反思、表达；

  • 可以培养自己观察和感受的习惯，让每一天都更有痕迹！

    ……

写日记的益处已被大量书籍系统阐述，但与其研读理论，不如直接提笔实践。最重要、最关键的事情，永远是先开始写！

只有我们开始去写、并且能够持续的写一段时间，才会发现一些针对自己的个性化的问题——晚上写会影响睡眠！写太多会耽误时间！用电脑写更好保存和浏览！有没有更好用的模版一打开就可以写得那种……

这些问题提前去了解其实用处不大的，就像我现在一下子告诉你 100 个有用处的思维模型，你大脑会告诉你它们很有用，但你的大脑同样还告诉你——我无法短期内都运用自如！自己没有踩过的坑，大脑永远难以建立强有力的神经联结，自然对某些重要但没经历过的东西没那么敏锐、印象深刻。

所以，最重要的是开始去写！问题会慢慢显现，然后再慢慢解决。

那么，日记可以写什么呢？

我觉得从写一件简单的事情开始，然后从事情-过程-结果-感想这几个细节去展开！这里面可以锻炼到自己描述事实、表达感受、反思结果这些能力，不是每个人都具备的。

  • 写自己今天印象深刻的一件事——开心的、难过的、感恩的……；

  • 写自己今天头脑里闪过的一个想法、一个思考，描述清楚；

  • 写自己今天完成的有成就感的事情，积累自信；

  • 写自己今天看到了的绝美画面；

  • 写自己收到的夸奖，更理性全面的认识自己；

    ……

写日记一段时间后，会有许多小问题显现的——效率上的；排版上的；体验感上的……

经历了这些阶段，我发现写作不光是写内容，还需要有好用的工具和平台来支持创作过程。所以，接下来我想聊聊我用过的写作工具和平台。

03 写作工具分享

前面提到：如果我们想写一点东西，可以从写日记这样一件小事情做起，目的就是降低开始写作这个行为的阻力和难度，而后在这个过程中慢慢的去发现一些问题，再慢慢去解决。

今天的话题，应该是很多写作者都会遇到的问题——有哪些好用的写作工具！

虽说 AI 的加持让非常多原本就十分优秀的笔记软件更加的优秀，市面上主打各种独家功能——block 块、双链、无限画布、大纲、AI 辅助……——层出不穷、各有特色。

但我还是秉承着那句话：事情应该力求简单，但不应过于简单！

做事情力求简单，主要是能够把注意力最大化的放在所做的事情上，减少流程、仪式、过程…对主体任务的影响；

但为什么不能过于简单呢？同样的原因，过于简单很多时候会无限趋近于简陋、不完善！

好比于我们想去一个地方吃饭，步行是最简单、无成本的，但开车过去看起来更能让人接受。

所以，接下来我给大家分享的工具，有的很简单，直接上手就可以；有的需要一些学习成本，大家不要看到简单的就说“太简单了，我不要用”；也不要说“太复杂了，我不要学”，这样的态度很不正常。就像学会了开车，可以让我们未来每一次吃饭游玩，来得更高效。

Markdown 语法

我第一个要给大家介绍的，也是最颠覆大部分人的东西，就是一种写作语法——markdown！

这种语法不同于传统的 word 编辑、记事本等等富文本软件，markdown 更能让我们专注于写的动作，淡化美化这个行为。

常用的 markdown 语法：

  • # + 空格 + 内容：可以表示几级标题，一个 # 级别最高、依次递减

  • - + 空格 + 内容：可以生成项目符号

  • 数字 + . ：可以生成有序列表

  • ** 内容 **：可以让内容粗体显示

  • * 内容 *：可以让内容斜体显示

  • -–：可以生成一条横线分割内容
    ……

这些常见的语法，就可以让我们生成一篇排版挺不错的文章了。

左侧是编辑模式，右边是预览模式——最终呈现的效果。

灵感软件

灵感总是转瞬即逝的，那么需要我们及时的捕捉记录，我推荐少楠的 flomo ，全平台同步，支持所有主流浏览器的扩展插件，几乎包含我们任何记录灵感的场合——在外面，随时随地用手机写下来或者语音说出来；在浏览网页时，在扩展插件里也可以随手一记，非常的丝滑。

而且这个软件，也是我认为用来尝试卡片笔记的好伴侣，非常的好用，值得推荐。

写作软件

我推荐 Typora ，这个软件的设计很简约大方，同时，是我用过的最好的及时渲染软件，写作体验非常的丝滑——所见即所得（编辑的同时可以看到呈现效果）的感觉真的很棒。

并且后期可以搭配图床软件 upic 或者 picgo 来嵌入永久图片；搭配 pandoc 更好的给自己的文章打标签、做引用资料、做导出。

管理软件

写的东西一多，就需要管理，我推荐 obsidian 这个软件，它的编辑体验没有 Typora 那么好，但是它的管理功能非常的强大，可以在插件商城里，根据自身的需求，安装各种专业化的插件，实现各项功能。比如一键导入微信读书的笔记、一键导入 flomo 软件里的内容、搭配 todo 类型的插件做任务管理、搭配思维导图插件给文章理清脉络……

总之，这是一款瑞士军刀级别的 All-in-one （多功能一体集成）软件，你能想象到的笔记模式、功能，它都可以做到，并且个人功能永久免费。

这个图谱功能我也很喜欢，可以展示出我们做过的笔记之间的关系，非常的有成就感，也非常实用——对于积累、联结素材非常的有帮助。我个人对于双链功能运用比较少，所以连接比较少。

润色网页

Markdown.com.cn 这个网站，可以让我们在自媒体平台发布内容前，再做一些美化、润色，因为这里面有一些基于前端 html 和 css 的主题，挺精美的！尤其是对于公众号这种不支持 markdown 编辑的大平台，在这上面润色一下再发布，是很不错的选择。

工具分享就到这里！说到底，适合自己的工具会极大提升我们的写作体验，但内容发布出去收获的正反馈，对于我们的激励是其它东西无法比拟的，下面和大家分享一下让自己内容更容易有人看的小经验。

04 我写得你更爱看

太阳底下无新事是一个古老的哲学概念——意味着这个世界大多、甚至全部的东西都是曾经的再现、一个又一个轮回——其实也从某种意义上有点悲剧色彩：我们无论怎样生活，走过的大抵都是别人已经走过的路！

但这又何妨，这并不能阻挡我们每个人书写属于自己的七彩剧本——讲述好自己的故事。

我发现一个有意思的现象：讲述好一个道理、一个概念或者一个规律，最有效、最严谨的表达手段是文字，最有效的获取手段是阅读！但为什么我有些时候更愿意去看某些内容非常相近、但人物性格不同的视频；还有不同写作风格和排版风格的博客……甚至接触这些东西的喜悦，要高于看书！

原因很简单——当道理和拥有不一样经历的主人公擦出火花，便成了故事；人类这种动物，大抵是不喜欢被说教的，却很少有不喜欢看故事、听故事的。

想让我们写出来的内容，有更好的「收视率」，请试着把自己当作主人公代入，像讲故事一样，把想表达的东西通过故事的形式呈现出去，效果会好得多。

我很喜欢《围城》这本书，它里面有太多太多的形容和类比**，让我们很容易对某种有距离感的新鲜事物，一下子产生理解并留有印象，这让我们本就在讲述故事的小说书籍中，进一步的拉近了距离，更加的沉浸投入**。

我也很喜欢一个问题：为什么余华是比罗翔老师伟大的多的作家，但看他们的采访总觉得罗翔老师要睿智一些？余华老师谈吐要“笨拙、随意”一些？

原因就在于罗翔老师本就是个哲学领域的忠实学徒，再加上他自己录视频分享观点的缘故，自然很多时候面对问题，反应和语言逻辑组织会迅速严密很多。但余华老师不是，他一定也是反应迅速且抓住要害的，但他优秀小说家这个身份，注定使他不会像罗翔老师那样去表述——他要用故事的形式去讲述自己对于许多事情的理解。这样一个短时间内讲出一个好故事的工作体量可是比直白的阐述清楚一个观点要庞大复杂得多。

所以，想把东西写清楚、让读者产生共鸣、有关注量，多加入一些自己的经历——讲好这个故事！

加油，那些和我一样爱记录、爱分享的人。

05 写作平台推荐

写的东西多了，会油然而生一种分享欲，并且愈发浓烈——感觉自己写的东西变得可读性高了、有内容有逻辑有深度有乐趣了，那么哪些平台值得我们去长期坚守、投稿内容呢！下面我会给大家几个甄选原则，并且推荐一些我目前正在坚守的平台。

不推荐的平台

有些平台，它的定位就不是让我们专注于文字内容编辑和分享，比如以视频播放为主以及以图文内容为主的平台（包括但不限于：小破站、dy、xhs），它们有流量，但不适合文字内容生存。

不佳的文字编辑体验的平台：比如 xhs ，你去看看它的文字编辑体验有多糟糕——字数限制、排版以及显示形式……，都注定了这个平台很难产出由文字为主体的优质内容。

无厘头的审核机制：有时候我们好不容易把想分享的内容编辑好了，但发布出去的时候，发现因为违反某些规则限制**，仅自己可看了，那种感觉真的太让人绝望了。存在这些不健全的审核机制的平台，也不要去接触。

……

还有一些其它的标准，我没有一一列举出来——反正就是让我们分享内容这个行为阻力太的的平台，我们可以果断放弃。

推荐的平台

首先：简书这个平台，我肯定是要推荐的，我非常喜欢它的侧边栏管理文章的功能，非常的简洁、清晰；而且，这个平台本身就是个 markdown 编辑器，可以在设置里把文本格式调成 markdown 后，一边编辑、一边预览最后的效果；它的大部分内容都是文字，里面有非常多有意思的内容，我们可以分享自己的内容之余，在里面接触其他的优质内容。

其次：知乎、少数派是我非常推荐的两个平台，有着绝佳的 markdown 编辑体验（尤其是少数派），而且里面的内容更加长、更加优质（前提是我们有自己对于优质的判断）。我更推荐我们在这两个平台上发布一些长内容、有一些深度的内容，因为看这两个平台的读者也是想找这些内容的人。这样做，会对我们自己有个很好的提升，也可以让自己收获一波粘性很高的忠实读者。

最后：公众号是我推荐的一个平台，如果只推荐一个，我会推荐这个！！为什么，其实也很简单——背靠中国基数群体最大的国民软件，只要内容够好、持续更新内容，根本就不用担心流量问题；并且，它的审核机制虽说不是绝对的自由，但比其他平台成熟太多太多，不会随意限制内容；而且它也比较方便我们去传递、分享自己的好内容……更容易让自己在这个过程中收获反馈感。它的缺点我今天就不提了，因为可以通过其它的方法解决。

提醒

写东西，有两件事情很重要——写和好好的写。

持续不断的去写是重要的，这是一切的根基。但好好的写也很重要，决定着我们写的东西能到达怎样的深度。

好好的写，我今天只想提一点——不要成为一个工具控和平台控——不能让除写东西以外的时间占比太大；为内容增加太多图片、视频、排版美化等等，没有这个必要——喧宾夺主、过犹不及！！只需要确认 3 种及以下的主题配色、确认标题级别和内容主体的字号区别，非必要不用图片的这些小原则，就可以了。

06 写东西的流程

最近这段时间，我基本恢复了日更。当然，是在其它平台！

倒不是说我要每天都写出很高质量的东西，而是把一个行为固定下来，会让写点儿东西这个行为启动变得不那么难、不那么挣扎！

而且，每天写点东西，让自己多多少少总可以思考点什么，把一周左右的小篇幅内容，周末整合、润色一下，就可以成为一篇有逻辑、有观点、有剖析的可读性不错的长文了。这些文章我会发布在其它的同名（胡拉图的后花园）平台（公众号、少数派、简书、知乎、个人Blog…），让更多人看到！

关于写东西，我自己折腾了这么些年，文笔、个性化的这些东西暂且不谈，但实实在在的在优化适合的写作流这件事情上确实是时刻都在迭代，契合自己。

我一定可以一直写下去！！给自己加油。

写东西的话，不同阶段，其实有不一样的流程。拿我自己来说：

第一阶段：21 年刚开始写东西是在公众号上，总喜欢写一些自己比较有感触的话题、不一样的经历、社会热点等等内容。

那时候写内容，基本就是一口气写到底的方式——脑海中有了个想法想要分享，然后就开始抓住这个灵感码字，最后会产生一篇还算有内容的小短文，但并不成熟——主题性、逻辑性、可读性，都比较一般。而且刚开始发布内容，热情很高涨，几乎是日更，所以也没那么多时间打磨和精进内容。

第二阶段：其实是日更的热情褪去的时候，好吧！其实更准确的说，是脑海里没有什么灵感和话题让自己有想要去写点什么的冲动了。这段时候，有意识的不去追求日更，这便是一个很大的转变——写东西和很多东西一样，不刻意的兴趣和爱好，其实才更能让自己享受和别人接受！

这段时间我做了几件事情：

  • 更加严苛的挑选自己的信息源：好的书、好的博客、好的 newsletter ……一定要精品；
  • 有意的优化自己的灵感记录、主题积累和写作流程：遇到好的东西怎么积累下来；
  • 存量到增量的转变：现在看的很多东西其实都是为了怎样更好把过去懂得的东西表达的更透彻、更科学；
  • 学习了一些编辑器的使用：让自己的内容有更好的阅读体验。

综上，有了这段时间的沉淀，我发布内容的数量少了，但内容的深度、逻辑感、可读性、排版比过去要好得多，在阅读量、关注量上，在当时也产生了一些让我感到开心的文章，算是很强的正向反馈了，超级开心。

最后阶段：是我 23 年考研备战的那一年，刻意地减少了想法、观念、认知的输出，更多的转向了生活、细节、画面的描绘——就是真真实实的把一周、一个月的有趣经历描绘好，最好达到能产生画面感的那种文字感受，然后加之一些自己的小思考、自我剖析，这样写出来的东西接受度、认可度会好很多。从数据和关注量反馈来看，更让我坚信是这样。

那段时间我的更新其实是挺稳定的，文章类型也开始有了许多种类的试探——日记、周记、月小结、书籍分享、好听音乐、电影观后感、科学小知识……，都让我尝试了个遍，也感受到了写不同类型的文章，有什么不一样的细节、技巧需要注意，有哪些需要重点突出的地方需要用不一样的手段去表述，是的！我又成长了一次。

正因为有这几个阶段的沉淀和改进，23 年考完研开始，我的写作流程、风格逐渐形成，带有个人标签——你只要现在看过我的文字，再去其他平台随意看一看内容，遇到匹配的风格，你可能都会一下子认出那就是我——就是这么的有标签！

希望那些喜欢用文字记录和表达自己的我们，可以慢慢来！

07 为了什么而写

我们因为某些推动着自己的力量、但实际不能准确说清楚究竟是因为什么而开始写作、分享的时候，如果这股暗力量褪去，我们继续写下去要靠什么？就会成为一个主要矛盾、一个等待认真思考的课题！

我为什么而写？

说实话，最开始停留在想写点什么东西的时候，幻想了几乎一切美好——有源源不断的灵感、有每天规律的写作时间、有精美的排版、让人兴奋的阅读量和粉丝数、可以产生自己的影响力、赚取一些额外收入……

这就是我还在想我可以写什么的时候，脑子里的真实想法。总觉得很多东西之所以没有收获好的结果，是因为自己还没有开始，只要开始了，就会做得好、收获美好的果实！

但究竟是完美主义作祟——隐隐觉得自己可能做不好、没准备好才不去开始；还是没有开始导致没有结果，是个值得博弈的话题——这个答案只有自己知道，但社会学的研究和表态告诉我们，大部分时间真的是完美主义作祟，让我们不敢开始。

我开始真正的写东西后，发现把一些东西写出来很爽、很放松，这是最直观的第一感受。起码在把自己想表达的东西写完最后一个句点、伸懒腰的时候，那种满足感会释放大量的多巴胺——这个多巴胺，当时我总以为是对我总算写完了某些东西产生的奖励，但其实是对这些内容发布出去可能会收获好的关注的期待——多巴胺给人带来快乐的本质其实是期待的这个过程，反而不是结果出现的片刻。

这样一种对于未知的一种偏向好的、美满的期待，支撑着我写了很多的内容。不好评价这样一种情况下写的东西好与不好，起码外在表现上来看——那段时间的我，是热爱记录、享受分享的。

再到最后，其实是考研的 23 年，我对于发布出去的每篇内容还有很高的期待嘛？要说有也是有，但是远没有开始那么浓烈了，那维持着我继续写下去的动力是什么呢？或者换句话说：这个时候我是为什么而写？

实话实说：要说真的是为了热爱而写，我觉得很苍白、没有太强的说服力！因为我已经说过了，这样一种热爱已经被那种满怀多巴胺加成的期待一次次的打破而消磨的差不多了，所以这时候支撑着我走下去的，我负责任地说，不是热爱！甚至有时候写东西不再是享受的，反而是感觉很长时间没有发内容了，再不写点什么就不太好，像是一种无形的监视和责任。

我为了什么而写：可能是为了向自己和读者证明，我是个爱写东西的人！然而这个证明并不那么容易和直观。

如果有人问你：究竟怎样才是爱一个人？！我想很多人都会有同一个感受——好像懂、好想会做…，但说不清楚！其实这样很正常，因为爱一个人这个东西并不是有「固定标准」的东西，而向自己或者别人证明一件事情需要符合很多人的标准、规则、期待！

虽说很多事情没有标准答案，但靠近标准答案的回答还是值得参考的——寻找强关联的表征去观察！

爱和性在现代社会中被无数次的证实是关联性非常高的一对组合，一对互相有爱的人，在性生活上大概率和谐美满；相反，就会差点意思。当然，这只是简单的充分必要条件的关系，可以从侧面反映出一些端倪，但不绝对。

我的自证也是：我首先给自己贴上了一个自愿贴上的标签——我开始了写东西、写了很多东西、积累了一些读者、我一直在说写东西去记录分享是很美好的……，所以我要去自证的话，最强关联的就是更新频率稳定、内容质量保证！

因为我内心对写东西究竟是怎样的看法，很多时候我自己都很难描述的清晰、准确，又怎么让你们相信我的真实想法呢？但看一些强关联的外在表征——一个写了很多年的人、一个挺真实的人、一个不断自我剖析的人……——大概率也会是一个热爱、享受写作的人！往往会靠近那个标准答案，你们说呢？

所以，如果后面有人问我，眼下的自己是为了什么而写作：不高姿态的讲，为了吸引关注、为了流量、为了收入…都对，但不全面！为了自证我爱写作的身份，也向读者证明写作的益处，才更贴切吧！

工具和平台虽然重要，但说到底，它们只是辅助。写作最核心的还是那些让我们动笔的想法和灵感。那么，灵感从哪儿来呢？下面我分享一下我的经验。

08 写作灵感来源

无论你是个想写东西分享的人，亦或者是个普通人，建议大家都创建一个自己的灵感库——你可能能笃定自己不想写作、也不想这样去分享日常…，没错，我相信！但每个人都不能保证自己是没有灵感乍现时刻的，当我们有了这样的时刻，把它们记录下来，是一件很有趣的事情，我相信没人不喜欢有趣！

在我之前提到的软件里（obsidian、typora、flomo）任选一个（更推荐flomo），创建一个灵感库文件夹或者是灵感标签，这样的话无论我们什么时候有了一些想法，就可以随时掏出手机，用文字、语音或者拍照片的方式先简单记录下来，后面看到这部分记录的时候，有兴趣或心血来潮把它们记录、描绘完整的时候，将会非常的享受。

我们有了灵感记录的手段后，其实更便于我们产生灵感的，是海量的输入小范围内的知识——有点像刻意练习，在这儿，我暂且叫刻意累积！

我积累灵感和素材的时候，有两种模式：

最常用的一种是发散式：我这周也不知道想写点什么，但是每天爱看点东西的习惯是很难改变的。所以在我散漫浏览信息的过程中，比如看到了一篇关于如何搭建个人博客网站的文章，我非常的感兴趣，那么可能我就会去其它的技术博客和平台上检索有没有相关的内容，以便于我自己可以去实操、完全的搭建好一个博客平台，也便于我向别人分享如何拥有一个自己的博客网页的内容和经验。

第二种模式是聚焦式：更实用、更高效的一种方法，我脑海中有一些很想分享的主题内容时，比如说「在校生如何科学的生活化减脂——饮食篇」、「在校生的每日EDC（Every Day Carry——每天携带的物件）」、「关于自媒体写作的一切」……我会先给它们细分成几个小版块，然后这一周每一天都往这些小版块里去输入相关内容，去吸收、总结、内化、输出一点自己的内容——可以是对于输入内容的总结、汇聚；也可以结合自己过往经历、经验去剖析融合。最后每个板块的内容都完善了，再把它们组合成一篇长文，注意逻辑、过渡。

以上就是我自己常用的两种积累灵感和素材的模式！

那么我们浏览、学习、输入的东西应该从哪里来呢？

这次我就不做具体的推荐了，因为每个人感兴趣的东西不一样，我只讲大的原则方向。

  • 善用谷歌搜索、谷歌学术，摒弃一切国内的搜索引擎；

  • 很多书不看，不看很多书，书不是很多、看不了很多书；

  • 精选一些个人博客，写的东西有出处的、有逻辑的；

  • 看文献，除专业科研要求外，多看高引文献，多看歪果人写的英文文献；

  • 看视频学会适应长视频，学会听视频，学会看完知识类视频做记录；

  • 闲暇轻松时，可以听一听播客（podcast），选自己喜欢的领域，选权威的人物；

    ……

    每个类里，别超过 3 这个数字，要不然会分散精力。

大家可以从这些内容里，寻找自己觉得合适的一样或几样，结合起来去丰富自己的灵感来源，并且去优化一下自己的生活流程，相信对大家有帮助。

但我还是要提 3 个小 tips：

  1. 别收藏、别收藏、别收藏！觉得有用的内容，就耐心看完；觉得可能有用或者没用的内容，就关闭。

    就算你可能有真的觉得很好的内容，但真的太长，一次性看不完，那也把内容 copy 下来在一个编辑器里，做简单的删、减，方便后期再读，这样也比直接收藏要好得多。

  2. 别 copy 、别 copy 、别 copy ！做总结、做笔记的时候，千万别直接用别人总结好的内容来代替自己——我知道那种看到别人写出来的东西就是我想表达的意思的契合感，但直接复制抄袭的行为一定会是我们的很多能力退化！一定坚持自己去描述，可以参考别人写的，从模仿开始，而不是 copy 、代替。

  3. 记得去整理灵感库！灵感之所以被记录下来，是因为我们对这个想法有感觉——这是不一样的东西，起码于我自己而言！但只是被记录、不被整合，那就是一堆被遗忘在垃圾堆里的小垃圾，毫无价值。

总的来说，写东西是个综合性的活动，需要明确动机、优化流程、选对工具、不断寻找灵感……，但这些标准都是个性化的！通过反思为什么写、改进写作习惯、挑选合适的工具、积极获取灵感，我们能写得更好，也更享受写作的过程。

希望我的分享能对你有所帮助，祝大家在写东西的路上越走越远。
